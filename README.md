# 🍫 Power BI Sales Performance Dashboard

## 📌 Overview
Welcome to the **Chocolate Sales Performance Dashboard** built using **Power BI**!  
This dashboard provides an interactive and data-driven overview of sales performance across different regions, time periods, and salespersons.

It helps businesses:
- 📈 Track sales growth
- 🌍 Analyze country-wise performance
- 🧑‍💼 Monitor salesperson productivity
- 📦 Measure operational efficiency
- 🎯 Support strategic decision-making

---

# 🚀 Dashboard Highlights

## 📊 Key Performance Indicators (KPIs)

| KPI | Value |
|---|---|
| 💰 **Total Sales Amount** | **28M** |
| 📦 **Total Boxes Handled** | **2M** |

These KPI cards provide a quick snapshot of overall business performance.

# 📈 Sales Trend Analysis

## 🗓️ How Much Amount by Year?
The dashboard includes a dynamic **Sales Trend Line Chart** that visualizes monthly sales performance over time.

### 🔍 Insights You Can Identify
- 📈 Peak revenue months
- 📉 Sales decline periods
- 🔄 Seasonal sales patterns
- 📅 Monthly business growth trends

### ✅ Why It Matters
This visualization helps management understand:
- Business momentum
- Revenue fluctuations
- Performance consistency

# 🌍 Country-wise Sales Analysis

## 🍩 How Much Amount by Country?
A donut chart is used to represent sales contribution from different countries.

### 📌 Key Benefits
